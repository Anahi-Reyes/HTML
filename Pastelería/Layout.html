♥ Tipos de pastel

Nosotros contamos con tres diferentes tipos de pastel y estos varian según la cantidad de pisos que contienen:

Sencillo
-Este tipo de pastel es de un solo piso, por lo que únicamente puedes seleccionar un sabor para todo el pastel.
Doble
-Este tipo de pastel cuenta con 2 pisos, por lo que puedes seleccionar 2 sabores diferentes o el mismo sabor para todo el pastel.
Venti
-Este tipo de pastel cuenta con 3 pisos, por lo que puedes seleccionar 3 sabores diferentes o el mismo sabor para todo el pastel.
